# StateMotion v0.1 Alpha — Feature Scope

**Status:** Frozen for Private Alpha. Do not expand without explicit approval.

This document is the authoritative scope boundary for the v0.1 Alpha release.
Any task proposing work outside these lists is out of scope and must be
deferred or re-approved.

## INCLUDED

### Transform
- Position (A/B)
- Scale X / Scale Y
- Rotation
- Anchor
- Opacity

### Timing / Progress
- AToB, BToA
- AToBToA, BToAToB
- HoldA, HoldB
- Manual
- Alignment: ClipStart, ClipEnd, EntireClip
- Duration
- Delay

### Easing
- Linear
- Ease In
- Ease Out
- Ease In Out
- Custom cubic Bézier (x1,y1,x2,y2)

### Presets
- Library (browse / search / filter / sort)
- Apply
- Create from instance
- Import / Export (`.stmpreset`)
- Favorites
- Collections
- Starter presets

### Launch Hardening (this milestone)
- Central product version
- Capability / status model
- Diagnostics / About view
- Copy Debug Report
- Human-readable error messages
- Safe-write policy (validate → plan → compat → execute)
- First-run / empty states
- Alpha badge
- Installation docs, test checklist, release gates, host-status ledger

## EXPLICITLY DEFERRED

- Crop
- Masks
- Rounded corners
- Stroke
- Glow
- Shadow
- Motion blur
- GPU / CUDA / Metal
- Graph editor (interactive curve editor)
- Licensing
- Cloud sync
- Telemetry
- Automatic updater
- New transition modes
- New starter presets

## Version concepts (kept distinct)
- **Product version** `0.1.0-alpha.1` — release identity (this milestone).
- **Preset format version** (`SCHEMA_VERSION = 1`) — `.stmpreset` format.
- **Binding revision** (`BINDING_REVISION = 2`) — native/panel param bindings.
- **Parameter count** (`PARAMETER_COUNT = 25`) — expected native params.

Bumping the product version does NOT change the contract dimensions.
