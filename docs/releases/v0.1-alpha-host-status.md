certification checklist. **Execution blocked in the current environment:**
- No C++ toolchain present (no MSVC / clang / gcc) → the real native `.aex`

  (Steps 6–26: native discovery, manual transform, easing visuals, undo,
  persistence, etc.).

**Automated evidence that DOES exist (SDK-free, run in this environment):**
- 23/23 panel suites PASS.
- `node tools/generate-contract.js --check` → CHECK OK (25 params, rev 2).
- Panel typecheck + production build PASS (product version `0.1.0-alpha.1`).
- Easing math parity: CUSTOM `(1/3,0,2/3,1)` and EASE_IN_OUT both equal
  `3x²-2x³` within 1e-6 (TS production evaluator + dedicated regression).
- Safe-write policy: zero `writeLogical` calls on incompatible/older/newer/
  no-effect contracts (unit-tested).
- Debug report sanitization: no paths/usernames/clip names/preset contents
  leaked (unit-tested).

**All host-dependent gates remain NOT YET VERIFIED.** No host gate was marked
PASS. Certification must be completed on a machine with the Adobe SDK + a C++
compiler + Premiere Pro, following `v0.1-alpha-test-checklist.md`.
